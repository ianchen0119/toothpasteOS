# 牙膏 OS

以 mini-riscv-os 為基礎，預計添加以下功能的作業系統:

- [x] Create and destroy the Child process.
- [x] Virtio supported.
- [x] Modify the interrupt vector (為了支援除了 Timer interrupt 以外的中斷).
- [ ] PLIC supported.
- [ ] Shell.
- [ ] Spinlock.
- [ ] Implementation on Disk and FileSystem.
